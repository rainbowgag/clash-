# Clash 分流配置 & Dialer 链式节点工具

本仓库用于分享 **Clash 分流配置示例** 以及 **Dialer 链式节点前端工具**，主要面向希望理解和使用分流、链式代理逻辑的用户。

---

## 一、Clash 分流配置说明

- 本配置 **主要适配 OpenClash**
- 建议在 **OpenWrt / OpenClash 环境** 中使用
- 其他 Clash 客户端（如 Clash for Windows、Clash Verge 等）**未做完整测试**
  - 理论上可兼容
  - 若存在兼容性问题，需用户自行调整

> 本配置为 **通用示例配置**，适用于大多数常见使用场景，目的是帮助用户理解分流逻辑和规则结构，而非针对特定网络环境的定制方案。

---

## 二、Dialer 链式节点工具说明

- 本项目包含 **Dialer 链式代理的前端工具**
- 使用方式：
  - 下载后在本地运行
  - 无需上传节点或订阅信息
- 适合需要：
